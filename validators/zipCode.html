US zipcode

This validator supports 4 digits US zipcode. If you want a valid US zipcode to be 5 digits exactly, use the regexp validator:

{% highlight javascript %} $(document).ready(function() { $('#postcodeForm').bootstrapValidator({ fields: { postcode: { validators: { regexp: { regexp: /^\d{5}$/, message: 'The US zipcode must contain 5 digits' } } } } }); }); {% endhighlight %}
